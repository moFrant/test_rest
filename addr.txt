<pk> - первичный ключ

   /api-token-auth/ - доступен всем, служит для получения токена, принимает два параметра username и password
    /country/ - для получения списка доступен всем, для добавления доступен админу. Служит для вывода/добавления стран
    /country/<pk> - для получения одной записи доступен всем, для изменения или удаления доступен для админа
    /sity/ - аналогично со странами, только выдает список городов
    /sity/<pk> - аналогично со странами, только операции с городами
    /category/ - для вывода списка доступен только для авторизованных активных пользователей, для добавления доступен для админа. Служит для вывода и добавления категорий
    /category/<pk> - вывод одной категории, изменение и удаление. Для вывода доступен для авторизованных, для изменения и удаления для админов
    /post/ - выводит все посты, а так же добавляет новые. Доступен для авторизованных пользователей
    /post/<pk> выводит один пост, доступно для авторизованных пользователей, изменяет и удаляет, доступно для автора и для админа
    /post_category/<pk> - выводит посты по категории, доступно для авторизованных пользователей
    /profile/ - получение списка пользователей, доступно для авторизованных пользователей. Тут же выводит списки избранного
    /profile/<pk> - получение одного профиля для авторизованных, изменение и блокировка для самого себя
    /users/ - выводит список пользователей с подробными данными, доступно для админа
    /users/<pk> выводит информацию по одному пользователю, можно изменять и удалять пользователей. Доступно для админа
    /images/ выводит все картинки, тут есть ссылки на оригинал, на уменьшеную копию и на превью
    /images/<pk>
